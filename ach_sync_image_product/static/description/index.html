Import Images to other Odoo

It doesn't matter which version you want to migrate the images from.

Translated languages

Step 1:

After installing the module you have to go to the menu item called technical>Sync Image Product.

Step 2:



You will have to place the data from the Odoo database from which you want to obtain the images. VERIFY THAT THE ADDRESS OF YOUR PREVIOUS ODOO DOES NOT END WITH THE SYMBOL "/", IF IT IS, DELETE IT

Fill in all the fields that appear, then press the login button.



Step 3:

When you are logged in, set the reference field, this field will serve as a key between the two Odoo, use default code or product name, in the "SET FIELD NAME" field you must place the field where you want to save the image, try to use the same both in one Odoo and the other, normally this field does not change in all versions.



Step 4:

Press the Import Images button, depending on the number of images you are going to synchronize, so you will have to wait, after it is finished, verify that the image is already in your new Odoo.
